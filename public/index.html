üìö Common Issues and Fixes

  • Login loop or 401 after login
    Reason: Cookie domain mismatch or Redis session not reachable.
    Fix: Set SESSION_DOMAIN to your parent domain and ensure REDIS_URL is reachable (e.g., redis://127.0.0.1:6379).
  • Website upload fails
    Reason: Upload middleware missing (multer) or target path not writable.
    Fix: Install multer and ensure WEB_SITES_BASE exists and is writable by the portal user.
  • Provisioned site not serving
    Reason: Nginx vhost invalid or not reloaded.
    Fix: Validate with ‚Äúnginx -t‚Äù and reload Nginx; verify Host header resolves to your server IP.
  • SMTP send fails or submission (587) closed
    Reason: Postfix submission service not enabled or Dovecot SASL not available.
    Fix: Ensure master.cf has a submission block and reload postfix; verify Dovecot auth sockets exist.
  • Cloudflare DNS not created
    Reason: Zone not found for domain or missing API token/zone id.
    Fix: Provide CLOUDFLARE_API_TOKEN and correct zone; prefer using explicit CLOUDFLARE_ZONE_ID.
